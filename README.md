## Ejecución

Para ejecutarlo, acceder a la carpeta `Cube` e introducir en la terminal el siguiente comando : 

`python -m http.server`

A continuación acceder a cualquier navegador a la dirección :

`localhost:8000`

y disfrutar del juego.
